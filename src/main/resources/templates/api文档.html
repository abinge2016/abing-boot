简介

获取所有的标准科室

请求地址
环境 HTTPS
测试地址 https://openapi.guahao-test.com/openapi
生产地址 https://openapi.guahao.com/openapi
访问规则
规则 描述
请求方式 POST
请求参数加密 无
响应数据解密 无
公共参数
Header
参数名 参数类型 是否必须 描述
appkey String 是
用户秘钥管理中的AccessKey ID，具体参加访问控制
method String 是
平台接口名称，此接口为：guahao.health.standarddepart.allstddeptlist
timestamp String 是
UNIX时间戳，单位毫秒，需要与平台服务器时间差值在2.5分钟内，如：1605868401000
version String 是
版本，此接口为：2.0
product-code String 是
产品编码，此接口为：11739tx98
message-id String 是
由调用方生成，请求唯一标识，长度不超过36位；相同message-id 2.5 分钟内只能被使用一次
sign String 是
签名，详情见签名介绍
content-type String 是
application/json
content-md5 String 是
请求体为JSON且报文不为空时，对JSON报文体数据进行md5并转化为32为大写的字符串，用于加签时使用
请求参数
参数描述
参数名称 参数类型 是否必填 描述
param object 否 请求对象
weiyi-appid string 是 appId
weiyi-authtoken string 是 请求参数
请求示例 
curl --request POST 'http://openapi.guahao-test.com/openapi' \
--header 'Content-Type: application/json' \
--header 'appkey: 具体的AccessKey ID' \
--header 'method: guahao.health.standarddepart.allstddeptlist' \
--header 'sign: 对请求头和请求报文进行加签算法后的签名' \
--header 'timestamp: 1617169795442' \
--header 'version: 2.0' \
--header 'product-code: 11739tx98' \
--header 'message-id: e6f9b934-e3fb-4b16-ac75-491770b9cfeb' \
--header 'version: 2.0' \
--header 'content-md5: 请求体为JSON报文并且JSON报文不为空时，对JSON报文体数据进行md5并转化为32为大写的字符串，用于加签时使用' \
--data-raw '{
	"param":{
		"consultShieldingType":null,
		"sourceType":null,
		"noLimited":null,
		"consultLegalDept":null
	},
	"weiyi-appid":null,
	"weiyi-authtoken":null
}'
响应参数
参数描述
参数名称 参数类型 是否必填 描述
code int 是 响应码，0-成功，非0-失败
data object 否 响应业务数据
message string 否 响应信息
响应示例 
{

	"data":{
		"flag":null,
		"sort":null,
		"message":null,
		"stdDeptList":null,
		"authentication":null
	},
	"message":null
}
